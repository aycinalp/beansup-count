# Beans Up — Count Apps

Inventory counting apps for Beans Up Coffee (3 Houston locations).

## Live link

https://YOUR-USERNAME.github.io/beansup-count/

## Apps

- **Monthly Count** (`monthly.html`) — Full inventory, 245 items, with prices. For Aycin.
- **Daily & Weekly Count** (`weekly.html`) — For baristas. Daily milk check, weekly count, frozen items, order list, sends report to WhatsApp.

## How to update

1. Edit the HTML file
2. Commit and push to `main` branch
3. Wait ~1 minute for GitHub Pages to deploy
